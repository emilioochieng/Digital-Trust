How It Works: Cryptographic Timestamping

What is Cryptographic Timestamping?

We use blockchain-based timestamping to create an immutable record of when media was first seen online. This allows us to detect manipulated or out-of-context content.

  • 1. Media is hashed using SHA-256
  • 2. Hash is timestamped on a public blockchain
  • 3. Any future modification changes the hash → detected as fake
